# Explanation
Encorings are referring to the absolutes that exist in human culture. In other words, take all the human cultures which for a long time never had contact, you’ll always see some absolutes, things like weaponry, religion, language, or things like statues, burial sites or monuments. Every person seems to like these things, which is strange. Maybe these are the puzzle pieces to figure out why humans exist as a whole. [[The Great Cycle]] has [[The Pyramids]] pop up over the cycles again and again. If we reincarnated, and were being tested again and again, then maybe if we can figure out how to put all of this together, we can break [[The Great Cycle]].
